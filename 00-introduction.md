## What Is Linux?

Linux is an **Operating System**.

An operating system:

* Starts the computer
* Controls hardware (CPU, memory, disk)
* Allows users to run programs

Examples of operating systems:

* Windows
* macOS
* Linux

Linux is mostly used through **commands**, not mouse clicks.

---

## Why Linux Is Important?

Linux is used on most servers in the world.

Linux is used in:

* Data centers
* Cloud platforms
* Web servers
* Banking systems
* Enterprise applications

If you want to work as:

* System Administrator
* DevOps Engineer
* Cloud Engineer

Linux knowledge is **required**.

---

## Linux vs Windows (Simple Comparison)

| Windows         | Linux                |
| --------------- | -------------------- |
| Desktop focused | Server focused       |
| GUI based       | Command based        |
| Paid license    | Free and open source |
| Limited control | Full system control  |

Linux gives more control and stability for servers.

---

## One-Line Important Definitions

* **Operating System:** Software that controls the computer
* **Kernel:** Core part of Linux that talks to hardware
* **Distribution (Distro):** Linux kernel with tools and package manager
* **Shell:** Program used to type commands
* **CLI:** Text-based interface to control Linux

---

## What Is Open Source?

Linux is **open source**.

Open source means:

* Source code is free
* Anyone can see the code
* Anyone can improve the code
* No license cost

Because of open source, Linux is:

* Secure
* Stable
* Trusted worldwide

## What Is CLI?

CLI means **Command Line Interface**.

In Linux:

* You type a command
* Linux gives output

Example:

```bash
ls
```

This command shows files and directories.

Most production servers run **without GUI**.

---

## What Is RHCSA?

RHCSA means **Red Hat Certified System Administrator**.

RHCSA exam is:

* Practical
* Hands-on
* Command based

In the exam:

* No multiple choice questions
* Only real Linux tasks

RHCSA proves that you can manage Linux servers.
